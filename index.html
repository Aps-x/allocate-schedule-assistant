Help Section

A guide for using this website

Step-By-Step Guide:

  1. Navigate to your timetable on the Allocate+ website
  2. On the left hand side, click on a tutorial or lecture link
  3. In the main window you will see a table with your tutorial or lecture details, select the entire table and copy it
  4. Paste the unit information into the input box below
  5. Add a name for the unit and a color and press the submit button
